# Inbox

Drop files here for automatic processing.

## Tabs
Put tab files (PDF, GP5, GPX, Guitar Pro) in `tabs/` with this naming format:

```
Artist - Song.pdf
Artist - Song.gp5
```

Examples:
```
Megadeth - Tornado of Souls.pdf
Marty Friedman - Tearful Confession.gp5
```

Then tell Claude: **"process my inbox"**

## Videos
To learn from a YouTube theory video, just tell Claude:

**"Learn this video: https://youtube.com/watch?v=..."**

Add a brief note about what it covers if you want, e.g.:
**"Learn this video: [URL] — it's about modal interchange"**
